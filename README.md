<h1 align="center">Video Player</h1>

```bash
# clone repository
git clone https://github.com/mburuanthony/simpleVideoPlayer.git
```
minimal web video player

#### featuers
- loop : `l`
- remove loop : `o`
- menu : `shift`
- hide menu : `esc`
- mute : `m`
- unmute : `u`
- next : `n`
- previous : `p`
- pause : `.`
- play : `,`
- refresh page : `r`
- skip +30s : `>`
- skip -30s : `<`
- drag & drop from playlist to play
